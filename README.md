# java-cli-bazel-graphql-hello

## Description
A POC for a bazel graphql project.

## Tech stack
- java
- bazel

## Docker stack
- ol.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
